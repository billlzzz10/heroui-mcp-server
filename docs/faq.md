# FAQ

## General Questions

### What is HeroUI MCP Server?

HeroUI MCP Server is a Model Context Protocol server for creating and managing UI components efficiently.

### How do I get started?

Follow our [Quick Start Guide](quick-start.md) to get up and running in minutes.

### Is it free to use?

Yes, HeroUI MCP Server is open source under the ISC license.

## Technical Questions

### What frameworks are supported?

- React
- Vue
- Angular
- Vanilla JavaScript

### Can I use it in production?

Yes, it's production-ready with proper error handling and performance optimizations.

### How do I report bugs?

Create an issue on our [GitHub repository](https://github.com/heroui/mcp-server/issues).

## Integration Questions

### How do I integrate with React?

See our [React Integration Guide](integration/react.md).

### Can I customize themes?

Yes, use the [Theme API](api/theme.md) to create custom themes.

### Is there a CLI tool?

Yes, install with `npm install -g @heroui/cli`.
